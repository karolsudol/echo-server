# echo-server

A simple echo server written in Rust using Std.

## Usage

```bash
cargo run 10 
```


```bash
nc 127.0.0.1 8000
```

or

```
cd echo-server-tokio
cargo run
```

## Output

```bash
server starting 127.0.0.1:8000
server has a delay of 10 milliseconds
server listening 127.0.0.1:8000
```
